## YouTube Analyzer
Web application for personalized analysis of YouTube videos using neural networks.
The system extracts video transcripts and transforms them according to user prompts, allowing users to quickly understand the main ideas of video content.

## Features
- Telegram Authorization
- YouTube Channel Subscriptions
- AI-based Transcript Processing
- Prompt-based Text Transformation
- Crypto Wallet Integration

The application allows users to receive structured summaries, key ideas, and sentiment analysis of YouTube videos
